# WifiConnection-API
Es una api sencilla para poder controlar y usar tu wifi desde una app web local desde tu pc o celular.

## Como la uso?
La api la puedes usar libremente donde y cuando quieras, si no sabes html ni java script, puedes utilizar como base el index.html dentro de la carpeta public.

### **Nota**
La api trae en la rama "/" el index.html dentro de la carpeta public para no tener que ejecutar web y api por separado
